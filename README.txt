================Instruções de execução===============
Através do terminal do sistema operacional Windows, acesse o diretório TrabalhoPratico e execute o seguinte comando: 

>>python ModeloArquivo.py caminho_da_instancia

Por exemplo, para executar a instância I1.PTTOR-16, digite>

>>python ModeloArquivo.py Instancias_1_40/I1.PTTOR-16

--------------------------------------------------------
Desenvolvemos utilizando a versão 2.7 do python.


Obs:.
Encontramos uma diferença entre o arquivo Rio16.txt e os arquivos das instâncias PTTOR-16. Por isso fizemos uma modificação no Rio16.txt. Trocamos <Custo Fixo> por <Custo fixo>
